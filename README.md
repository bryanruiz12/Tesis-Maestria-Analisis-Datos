# Optimización de Portafolios: Markowitz vs. Deep Reinforcement Learning (PPO)

Este repositorio contiene el código fuente desarrollado para la Tesis de Maestría en Análisis de Datos.

## Contenido
* **Main_PPO.py**: Script de entrenamiento y validación del agente PPO.
* **Markowitz_Benchmark.py**: Script para el cálculo de la frontera eficiente y portafolios clásicos.
* **Data_Loader.py**: Módulo de descarga de datos desde Yahoo Finance (SPY, DAX, GLD).

## Librerías utilizadas
* Python 3.x
* Stable Baselines3
* Pandas, NumPy
* Plotly, Matplotlib
* Yfinance
